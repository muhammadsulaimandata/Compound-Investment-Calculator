#  Investment & Interest Calculator

A robust Python GUI application for calculating and visualizing investment growth. This tool allows users to calculate both Simple and Compound interest with customizable frequencies and contributions.

##  Overview

This project was developed as a final semester project to demonstrate proficiency in Python and GUI development. It helps users plan their financial future by providing detailed year-by-year breakdowns of their investments.

- **Dual Modes:** Calculate both **Simple Interest** and **Compound Interest**.

- **User Friendly:** Includes "Calculate" and "Clear" functionality for testing multiple scenarios.

##  Tech Stack

- **Python 3.x**: Core logic and calculation algorithms.
- **Tkinter**: Standard Python library used for creating the Graphical User Interface (GUI).
